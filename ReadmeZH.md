# 以撒的纸杯服务

《以撒的纸杯服务》是《以撒的纸电话》的服务器软件，现以开源的方式进行分发。为《以撒的结合：忏悔》设计，需配合《以撒的纸电话》插件使用。

[点我下载](https://github.com/frto027/IsaacPaperServer/releases)

# 版本号

版本号遵循格式`X.Y.Z`，其中：
- `X`是主版本号
- `Y`是次版本号，或者协议版本号。这个版本号必须和客户端的协议版本一致。
- `Z`是补丁版本号，随着开发进度频繁增加。
